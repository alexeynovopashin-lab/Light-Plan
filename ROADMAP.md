# Light Plan — дорожная карта

Прототип: `index.html` (один самодостаточный файл, без сборки и зависимостей).
Живая версия: https://alexeynovopashin-lab.github.io/Light-Plan/

---

## Сделано

**Главный экран «Сегодня»**
- Купол: дуга дня, время внутри, солнце тонет за горизонт, зарево гаснет золото → алый → розовый → синий
- Звёзды: проступают с −10°, полная сила с −18°, вращаются как диск день/ночь
- Отсчёт до ближайшего света + тренд облачности к окну (спарклайн)
- Сводка: экспонометр качества света, характер света, тени, небо, ветер
- Шапка: температура + иконка + состояние + мин/макс (как виджет Apple)
- Инженерный режим: спойлер с азимутами, сумерками, погодой; иконки у групп

**Управление временем**
- Единый закреплённый ползунок над таб-баром (общий для «Сегодня» и «Карты», не прыгает)
- Метка «↺ сейчас» появляется только при отмотке, тап возвращает
- Щелчки-детенты каждые 15 мин (Web Audio; на iOS Taptic из веба недоступен)

**Карта**
- Схема с траекторией солнца по азимутам, золотые окна, тени считаются из высоты солнца

**Планировщик «Съёмки»**
- Шапка: `‹ год` · добавить · поиск
- Календарь: точка = съёмка (нет съёмки — нет точки), цвет точки = какой свет достанется
- Просрочка сдачи: терракотовая подложка под числом, медленно дышит
- Сводка дня под спойлером (планировщик ведёт съёмками, а не погодой)
- Съёмки дня компактной строкой: время, клиент, статус сдачи
- Форма: жанры (настраиваются шестерёнкой), диапазон начало–конец, срок сдачи,
  желаемая погода с предупреждением, место, клиент, заметки
- Правка и удаление записей, поиск по клиенту/типу/месту
- Всё хранится локально (`localStorage`, ключ `lightplan.v1`)

**Сроки сдачи материала**
- Пресеты по жанрам: свадьба 90 дн., семья 14, портрет 7, репортаж 3, архитектура 5
- Режимы: по жанру / единый / без срока; переопределение у каждой съёмки
- Отметка «материал сдан»

   - «Начало / Конец» с инлайн-барабаном под строкой вместо шторки

## Дальше

3. **Клиент как сущность** — сейчас это строка текста. Нужен для истории, поздравлений, статистики
4. **Карточка клиента**: гонорар и расходы, ссылка на готовые фото (wfolio, Я.Диск),
   даты праздников с напоминаниями, история съёмок
5. **Бэкап**: экспорт базы файлом, выгрузка в своё облако через системный «Поделиться».
   Принцип: на нашем сервере не хранится ничего
6. **Статистика**: прибыль, съёмки по жанрам, отработанное время, среднее время сдачи
7. **Настройки**: часовой пояс, приватность, справка, поддержка, оценить
8. **Состояния экранов** из Screen Specification: Loading, Offline, Empty, Location Denied
9. Настоящая карта вместо схемы; светлая тема («утренний воздух»)
