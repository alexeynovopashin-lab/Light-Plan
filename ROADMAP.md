# Light Plan — дорожная карта

Прототип: `index.html` (один самодостаточный файл, без сборки и зависимостей).
Живая версия: https://alexeynovopashin-lab.github.io/Light-Plan/

---

## Сделано

**Главный экран «Сегодня»**
- Купол: дуга дня, время внутри, солнце тонет за горизонт, зарево гаснет золото → алый → розовый → синий
- Звёзды: проступают с −10°, полная сила с −18°, вращаются как диск день/ночь
- Отсчёт до ближайшего света + тренд облачности к окну (спарклайн)
- Сводка: экспонометр качества света, характер света, тени, небо, ветер
- Шапка: температура + иконка + состояние + мин/макс (как виджет Apple)
- Астро-режим: спойлер с азимутами, сумерками, погодой; иконки у групп

**Управление временем**
- Единый закреплённый ползунок над таб-баром (общий для «Сегодня» и «Карты», не прыгает)
- Метка «↺ сейчас» появляется только при отмотке, тап возвращает
- Щелчки-детенты каждые 15 мин (Web Audio; на iOS Taptic из веба недоступен)

**Карта**
- Настоящая карта (Leaflet + тёмные тайлы CARTO), только при сети
- Маркер = центр кадра; карта ездит под ним, место сохраняется и двигает весь свет
- Компас света поверх: путь солнца по азимуту, золотые окна, восход/закат, луч
- Часовой пояс — оценка от домашнего меридиана; «моё место» по геолокации
- Тени — честно: длина в ростах и направление (высоты зданий — будущий слой)

**Погода**
- Настоящий прогноз Open-Meteo (без ключа, CORS): облачность, температура, ветер
- Кормит модель качества света — дождливый день сам теряет золотые окна
- Синхронный интерфейс сохранён; офлайн и дальше 16 дней — тихий откат на мок
- `timezone=auto` даёт настоящий пояс места; источник честно подписан

**Прогноз заката**
- Балл 0–100 по трём ярусам облаков в час заката + влажность
- Нижний ярус — блокиратор (множитель), средний и верхний — экраны afterglow
- Ведёт категорию дня; наружу — словами и цветом точки, числа в астро-деталях

**Место**
- Задаётся с карты, геолокацией или ручным вводом координат на «Сегодня»
- Меняет свет и погоду для всего приложения; карта и «Сегодня» синхронны
- Имя места — обратное геокодирование: крупно город, мелко точное место
- Строка золотого часа на «Сегодня» — чтобы не спутать его с закатом

**Управление временем**
- Светило перетаскивается по дуге купола (солнце и луна) — прямое управление
- Слайдер остаётся для точности одной рукой; закатный балл вынесен в сводку

**Луна**
- Купол переключается между солнцем и луной двумя иконками
- Дуга проложена по пути луны, диск нарисован фазой: растущая освещена справа
- Фаза, освещённость, отсчёт до восхода или захода; в астро-режиме — группа «Луна»

**Планировщик «Съёмки»**
- Шапка: `‹ год` · добавить · поиск
- Календарь: точка = съёмка (нет съёмки — нет точки), цвет точки = какой свет достанется
- Просрочка сдачи: терракотовая подложка под числом, медленно дышит
- Сводка дня под спойлером (планировщик ведёт съёмками, а не погодой)
- Съёмки дня компактной строкой: время, клиент, статус сдачи
- Форма: жанры (настраиваются шестерёнкой), диапазон начало–конец, срок сдачи,
  желаемая погода с предупреждением, место, клиент, заметки
- Правка записей и удаление свайпом влево; удалённое возвращается шесть секунд
- Поиск по клиенту/типу/месту
- Экран года: двенадцать месяцев, число съёмок под именем месяца
- Недельный вид: семь световых лент — день, золотые окна и съёмки на шкале суток
- «↺ сегодня» и «↺ 2026» — появляются только при отмотке, как у ползунка времени
- Всё хранится локально (`localStorage`, ключ `lightplan.v1`)

**Форма съёмки**
- Группы вместо плоской ленты полей, ✕ и ✓ кружками в шапке
- «Дата · Начало · Конец» с инлайн-барабаном под строкой, длительность считается сама
- Жанр — пресет формы: у стрита и пейзажа нет ни сдачи материала, ни клиента
- Правка внутри жанра запоминается и в следующий раз перебивает пресет
- Жанры и погода — пиктограммы с подписью; срок сдачи — шкала с детентами
- Ползунок времени: гравировка рисок, пиктограммы восхода и заката вместо слов

**Сроки сдачи материала**
- Пресеты по жанрам: свадьба 90 дн., семья 14, портрет 7, репортаж 3, архитектура 5
- Режимы: по жанру / единый / без срока; переопределение у каждой съёмки
- Отметка «материал сдан»
