# Light Plan

Инструмент фотографа: прибор понимания света и планировщик съёмок.

**Прототип:** https://alexeynovopashin-lab.github.io/Light-Plan/

Открывается и на десктопе (в рамке устройства), и на iPhone. Для полноэкранного
режима: Safari → Поделиться → «На экран Домой».

## Что это

Две половины с жёсткой границей между ними.

**Прибор света** — экраны «Сегодня» и «Карта». Один вопрос: какой будет свет и
когда. Дуга дня, отсчёт до ближайшего золотого часа, тренд облачности, положение
солнца и длина теней на карте.

**Планировщик** — экран «Съёмки». Клиенты, диапазоны времени, сроки сдачи
материала. От календаря телефона отличается тем, что знает, попадает ли съёмка в
хороший свет и что вы кому должны.

## Как устроено

Один самодостаточный файл `index.html` — без сборки, зависимостей и сети.
Солнце считается по NOAA, погода пока имитируется детерминированно (один и тот
же день всегда даёт один прогноз). Записи хранятся на устройстве в `localStorage`.
